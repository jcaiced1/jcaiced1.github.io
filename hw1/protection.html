Basic protection checklist

  1. Use strong and unique passwords.
  2. Enable multi-factor authentication.
  3. Install updates as soon as possible.
  4. Back up important files often.

Defense in depth

Security works better when you combine technical controls and user habits.

Simple idea

If one defense fails, another layer can still lower the risk.
